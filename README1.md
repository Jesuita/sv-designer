# SV Designer

Motor de diseño editorial automático para Suban El Volumen.

---

## 🎯 Objetivo

SV Designer es una herramienta que permite generar portadas periodísticas profesionales en formato **1080x1350 (Instagram 4:5)** a partir de:

- Imagen
- Sección (categoría)
- Título
- Bajada

El sistema aplica reglas visuales fijas para mantener consistencia editorial.

---

## 🧠 Concepto

SV Designer **no es un generador creativo**.

Es un motor determinístico que replica el trabajo de un diseñador editorial, garantizando:

- Consistencia visual
- Identidad de marca
- Rapidez de producción
- Escalabilidad

---

## ⚙️ Stack tecnológico

- Node.js
- Express
- Puppeteer (render y export PNG)
- HTML + CSS + JavaScript (frontend)
